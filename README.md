# pizzas-shekinah
1. Install dependencies:    `npm install` 2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key 3. Run the app:    `npm run dev`  ## Imagens dos produtos  - As fotos ficam em `public/images/` e são servidas pelo Vite. - Cada produto usa automaticamente a imagem baseada no nome, em formato slug:   - Ex.: `Bacon` -> arquivo `public/images/bacon.jpg`   - Ex.: `Bacon Crocante` -> arquivo `public/images/bacon-crocante.jpg`   - Ex.: `Refrigerante Lata` -> arquivo `public/images/refrigerante-lata.jpg` - Formato recomendado: `jpg` (padrão). Se preferir `png`, ajuste o código em `constants.ts`. - Se a imagem não existir, o sistema mostra `public/images/placeholder.svg`.  ### Como trocar a foto de cada pedido  - Coloque a foto correta em `public/images/` com o nome do produto em minúsculas, sem acentos, separado por hífens e com `.jpg`. - Exemplo: para o produto "Calabresa", adicione `public/images/calabresa.jpg`. - Não precisa alterar o código: o app já busca pelo nome automaticamente.  ### Como adicionar mais produtos  - Edite o arquivo `constants.ts` na lista `PRODUCTS`. - Para pizza (categoria `Salgadas` ou `Doces`):   - Use `createPizza('id', 'Nome do Produto', 'Descrição', 'Salgadas')` ou `'Doces'`.   - Crie a foto correspondente em `public/images/<slug-do-nome>.jpg`. - Para bebidas: adicione um objeto com `{ id, name, description, category: 'Bebidas', isPizza: false, basePrice, image: imagePathFor(name) }`. - Após salvar, rode `npm run dev` e verifique no navegador.
